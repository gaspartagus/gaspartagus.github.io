trie.js

A cute little module for creating and searching trie hashmaps. It is designed for searching for sequences of words in a text.
For example the text "Hello, I'm John Coltrane" will be parsed as ["Hello"] and ["I'm", "John", Coltrane"] and hopefully, using a trie of artists, "John Coltrane" will be returned.

dictionnaire.js, along with artistRate.js

An array of 182 413 musical artists, sorted by name length and popularity, that is to say the amount of information available on it (number of websites, number of images and length of the description).

The original data is a monthly updated XML file from http://www.discogs.com/data/

This piece of code isn’t reusable but used to create a compact disposition of pictures in the DOM, looking like this :

It is done in 3 steps, first dispose the elements in order of size in several circles :

Each rectangle is symbolized as four dots in a plane. Add theoretical dots around the center block so that corners are found on first iteration. Then just move the blocks around to the nearest corner available.

To find which corner to move the block to, consider for each point of the whole layout all the points around. If one of the four quadrants around the point is empty and has edges, then it’s a corner.

Only the bottom right quadrant around the red point is a corner.

Color.js

A bling-bling effect for html video elements.

In canvas the current frame of the video element can be passed to canvas.getContext("2d").drawImage(video) and the pixels can then be manipulated with context.getImageData(). This piece of code computes the mean color of the down-sampled current video frame, changes its saturation and luminosity and displays it in the background of the page.

Find it wrapped as Chrome extension for Youtube here : https://chrome.google.com/webstore/detail/bling-your-tube/himnakdghmacgghflcinjdgjnihfimpa
